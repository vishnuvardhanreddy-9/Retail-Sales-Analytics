# 📊 Retail Sales Analytics Dashboard

An interactive **Retail Sales Analytics Dashboard** built using **Python, Pandas, Plotly, Streamlit, SQL, and PostgreSQL**. The dashboard enables users to analyze sales performance, monitor business KPIs, and gain insights into revenue, stores, products, customers, and payment trends through interactive visualizations.

---

## 🚀 Features

- 📌 Business KPI Dashboard
  - Total Revenue
  - Total Orders
  - Total Customers
  - Total Products
  - Total Stores
  - Average Order Value

- 📈 Sales Analysis
  - Monthly Revenue Trend
  - Revenue by Category

- 🏪 Store Performance
  - Top 10 Stores by Revenue
  - Top 10 Products by Revenue

- ⚙️ Operations Dashboard
  - Payment Method Distribution
  - Order Status Distribution

- 🎛 Interactive Sidebar Filters
  - Year
  - Category
  - Store
  - Payment Method
  - Order Status

- 📥 Download filtered data as CSV

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/Retail-Sales-Analytics.git
```

Navigate into the project:

```bash
cd Retail-Sales-Analytics
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the Streamlit dashboard:

```bash
streamlit run dashboard/app.py
```

# 📈 Dashboard Capabilities

The dashboard allows users to:

- Monitor business performance using KPI cards.
- Analyze monthly sales trends.
- Identify the highest revenue-generating product categories.
- Compare store performance.
- Discover the top-performing products.
- Analyze payment method usage.
- Track order status distribution.
- Filter the dashboard dynamically.
- Export filtered data to CSV.
